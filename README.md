 Offline Quran App (Mushaf)

A modern, luxury offline-first Quran application built with React Native and Expo.

 Features
- Offline First: Access full Mushaf content without an internet connection.
- Audio Recitations: Integrated audio playback using `expo-audio`.
- Custom UI: Beautiful typography and responsive Ayah badge markers.
- Cross-Platform: Built for Android using Expo Application Services (EAS).

 Tech Stack
- Framework: [Expo](https://expo.dev/) (React Native)
- Audio Engine: `expo-audio`
- Storage: `@react-native-async-storage/async-storage`
- Build System: EAS Build (Android APK)
